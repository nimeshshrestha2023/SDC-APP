C Programming

Welcome to the C Programming course! C is a powerful, general-purpose programming language widely used for system programming and creating applications. It provides low-level access to memory, making it a preferred choice for system and embedded programming.

Why Learn C Programming?

C programming is the foundation for learning other programming languages and plays a significant role in understanding how software interacts with hardware. Some key benefits of learning C programming include:

  • Improved understanding of how computers work at a low level.
  • Enhanced problem-solving skills.
  • Ability to work with high-performance applications and embedded systems.
  • Helps in learning other programming languages like C++, Java, and Python.

Topics Covered in This Course:

  • Introduction to C programming language
  • Data types, variables, and constants
  • Operators in C
  • Control flow (if-else, loops)
  • Functions and recursion
  • Arrays and strings
  • Pointers and memory management
  • Structures and unions
  • File handling
  • Dynamic memory allocation

    int num, reversed = 0, original, remainder;
    printf("Enter a number: ");
    scanf("%d", &num);
    original = num;

    while (num != 0) {
        remainder = num % 10;
        reversed = reversed * 10 + remainder;
        num /= 10;
    }

    if (original == reversed) {
        printf("%d is a palindrome.", original);
    } else {
        printf("%d is not a palindrome.", original);
    }

    return 0;
}

22. WAP to print Fibonacci series up to 'n' terms

#include 

int main() {
    int n, t1 = 0, t2 = 1, nextTerm;
    printf("Enter the number of terms: ");
    scanf("%d", &n);

    printf("Fibonacci series: ");
    for (int i = 1; i <= n; ++i) {
        if (i == 1) {
            printf("%d, ", t1);
            continue;
        }
        if (i == 2) {
            printf("%d, ", t2);
            continue;
        }
        nextTerm = t1 + t2;
        t1 = t2;
        t2 = nextTerm;
        printf("%d, ", nextTerm);
    }

    return 0;
}

29. WAP to check if a number is a perfect number

#include 

int main() {
    int num, sum = 0;
    printf("Enter a number: ");
    scanf("%d", &num);

    for (int i = 1; i < num; ++i) {
        if (num % i == 0) {
            sum += i;
        }
    }

    if (sum == num) {
        printf("%d is a perfect number.", num);
    } else {
        printf("%d is not a perfect number.", num);
    }

    return 0;
}
